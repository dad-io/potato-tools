# potato-tools 🥔

Potatoes are delicious and nutritious, and that old gear still has some vitamins and minerals. 

Soup it up!

- **[hud/](hud)** — a live desktop-wallpaper HUD that renders *behind* your icons (native GDI+, ≈0 VRAM). An LLM-inference panel (LM Studio on the GPU: VRAM ceiling, KV-cache projection, prefill/decode phase, CPU-offload spill) and a system-health panel, both fed by a tiny Python backend.
- **[potato-peeler/](potato-peeler)** — read-only system auditor.
- **[potato-launcher/](potato-launcher)** — reversible tuning pass + upgrade plan.
- **[potato-skins/](potato-skins)** — Rainmeter skins. `SystemHMI` (aka *daasktop*) is a glanceable right-edge system panel.

Built with tender frugality and care.
